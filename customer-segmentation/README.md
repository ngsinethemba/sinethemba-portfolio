# Customer Segmentation Analysis

## Overview
Applied RFM (Recency, Frequency, Monetary) analysis and k-means clustering to segment 397,884 retail transactions into four distinct customer groups.

## Key Findings
- Identified 4 customer segments: Champions, Loyal Customers, At Risk, New/Low Spenders
- Used elbow and silhouette methods for optimal cluster validation
- Provided actionable marketing recommendations for each segment

## Technologies
- R (tidyverse, ggplot2)
- K-means clustering
- Statistical analysis
