# MyBrew POS System Analysis & Design

## Overview
Comprehensive system analysis for a real-world coffee shop client, including database design, UML modeling, and requirements specification.

## Deliverables
- Entity Relationship Diagram (ERD) with 10+ normalized entities
- Use case diagrams and system sequence diagrams (SSDs)
- Functional and non-functional requirements analysis
- Data population strategy with referential integrity

## Methodology
- Stakeholder interviews
- Requirements elicitation
- UML modeling using Visual Paradigm
- Database normalization
